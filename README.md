# Noorani Browser

A premium desktop browser built for the Muslim community. Privacy-first. Open source. Free.

## What's inside

- **Prayer times** — calculated offline from your location, 8 calculation methods, both Hanafi and Shafi Asr support
- **Qibla direction** — accurate compass bearing to Mecca
- **Hijri calendar** — alongside the Gregorian date
- **Azan notifications** — silent by default, customizable per prayer
- **Offline Quran reader** — Uthmani script, Sahih International + Pickthall translations, Amiri Quran typography
- **Dua bookmarks** — separate from regular bookmarks, save verses with notes
- **Halal content filter** — 127,000+ blocked domains, user-controllable
- **Riba & gambling blocker** — for finance-conscious browsing
- **Ramadan mode** — auto-activates during Ramadan, blocks distractions during fasting hours
- **Privacy-first** — zero telemetry, zero tracking, zero data collection from the app

Plus everything you expect from a modern browser: tabs, bookmarks, history, downloads, themes, custom homepage, and more.

## Download

Get the latest release for your platform: [Releases page](https://github.com/WaleedNaeem/noorani-browser/releases/latest)

- Windows: Noorani-Browser-Setup.exe
- macOS: Noorani-Browser.dmg
- Linux: Noorani-Browser.AppImage

## Build from source

```bash
git clone https://github.com/WaleedNaeem/noorani-browser.git
cd noorani-browser
npm install
npm start         # run in dev mode
npm run build     # build installers for your platform
```

Built with Electron, vanilla JavaScript, and care.

## Privacy
